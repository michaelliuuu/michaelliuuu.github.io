PantryPal


PantryPal is a simple web-based application that utilizes the Gemini Pro AI API to create recipes based on ingredients you have on hand. Utilizes HTML/CSS as its frontend and JavaScript as the backend.

BlockWatch


BlockWatch is a Google Chrome extension that displays to users the optimal time to buy crypto. As well as, showing the current price and trend within the current month of Etherium and Bitcoin. Utilizes HTML/CSS as its frontend and JavaScript as the backend.

DIVVY


DIVVY is a web-based application that allows users to easily split expenses with others. Groups are created with friends and family to allow bills or payments to be one place. Ranging from vacation trips to a cup of coffee, DIVVY can track it all. The site was developed using Flask, a Python web framework, which serves as its backend. SQLite3 was utilized to store and manage the database. For the site's frontend, HTML/CSS was employed for its simplicity.

Final project for CS50x
